# TGIF Talks

Markdown talks and demos for Friday colleague presentations.
Most talks use Presenterm.

## Talks

- `cosmos-postgres-migration-plan/`: Disposable One-Time Code.
- `exposed/`: Warum mich `JPA`/`Hibernate` nervt und warum `Exposed` angenehmer ist.

## Working Pattern

- Keep source Markdown and supporting files in each talk folder.
- Put demo code or commands next to the talk when they belong to the presentation.
- Keep generated exports in an `export/` folder if needed.
- Extract only durable, reusable lessons into agent knowledge notes.
